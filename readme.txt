Для работы приложения требуется .Net Framework 4.7.2
Все необходимые для работы приложения пакеты предустановлены.
По умолчанию сервер разворачивается по адресу http://localhost:44364, в случае необходимости его можно изменить в свойствах проекта.
По умолчанию клиент подклчается к адресу http://localhost:44364, адресс сервера для клиента можно изменить в главном окне приложения.
Сервер создает локальную базу данных MSSQL с именем userstore, данные параметры можно изменить в файле Web.Config
Для работы с базой данных используется EntityFramework.
